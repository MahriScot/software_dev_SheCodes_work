Lesson 3 - Responsive Images

1. Without bootstrap

Use css in `style` to e.g. width: 50%; orrr max-width: 100%; etc. Also use height:auto; as this all will make things responsive. Using max-width and height:auto; will help with pixelation. The picture will stop growing with the page when quality is upset.


2. With Bootstrap

Using class= img-fluid
You can also make the picture rounded (or with a bigger radius = rounded-circle), have border with a small shadow, give it padding, a margin, etc.

Using images in a grid

use img-fluid to make it responsive, then grid to make it a lovely grid with the photos next to each other but ontop of each other if someone is on a phone e.g.. In that case, maybe always give the image a margin at the bottom so that they aren't stuck together when in a column rather than a row
