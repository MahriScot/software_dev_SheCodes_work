Lesson 1 - CSS Media Queries

e.g. When you want to center the h1 only on small resolutions (aka mobile devices)

you can use css - text-align: center
or put CSS Queries at the top of style.css using e.g. @media (min-width: 400px) {apply css}
but max-width is better e.g.
@media (max-width: 600px) {
h4 { text-align: left;
color: red; } }

this is an h4 and it will turn red, become centered, and bigger font size, with smaller resolution

it's quite good to use max-width 800 or 900px as it will cover most mobile/ ipad devices, but it depends what you're doing.
if you want to hide something on small devices:

this h3 is not super important so we might want to simply not show it on a mobile device so we can hide it in @media (max-width: 800px) {h3 {display: none;}}

you can change anything - font size, colours, etc etc etc

we can also use bootstrap - obviously download the appropriate CSS bootstrap link. and in the html we can have h2 class:"d-none d-sm-block" which means it won't display your h2 unless it's bigger than `small`
or you can have h2 class="d-block d-sm-none" which will hide what you have if the device is `small` or more

you know which order to do this in as you start with the smallest resolution option on the left (e.g d-block if you want it on your smallest resolution), and then go bigger
